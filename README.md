# Project
    This repository contains the code that is used for the mentioning functionallity of Zendesk.

# Getting Started

    git clone git@bitbucket.org:netvlies/mis-zendesk.git
    cd mis-zendesk
    composer.phar install

# Run the tests

    cd mis-zendesk
    phpunit
